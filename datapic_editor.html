What is a h2?

The h1 to h6 tags are used to define HTML headings.

h1 defines the most important heading. h6 defines the least important heading.
Why we use a h3 here?

Unlike a h1 a h3 can be used multiple times on a webpage as its job is to be the heading for a web page's sub topics.
What is a paragraph tag?

The HTML p element defines a paragraph. We have added styles to make the impact figure more prominent as it is the most important piece of data.

Why the impact figure and data description sit in the same div?

A div tag defines a division or a section in an HTML document. The div tag is used to group block-elements to format them with CSS. The impact figure and its discription are part of the same div as they are contextually related.
Whats is an unordered list?

An unordered list starts with the ul tag. Each list item starts with the li tag. A list can contain as many items as you like. The list items will be marked with bullets (small black circles), unless the css style attribute list-style-type: none is being added, as is the case with regards to the example data pic.

In contrast, an ordered list starts with the ol tag. Each list item starts with the li tag. The list items will be marked with numbers.
The data source and styling classes

Don't forget to indicate the source of your data!

How to change the background Image

1) URL:Upload your background image to your school's server (check with your teacher for details). Replace the 'background-image' URL with the path to your picture. When done, copy the code and paste it into your school's website to display your data pic.
